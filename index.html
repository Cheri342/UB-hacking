Password Generator

About

Hello, our group is made up of: Hrishikesh Vichore, Christine Cherian, Disha Patel, and Valerie Wong. For this year's hackathon for UB, our group decided to make a password generator that runs in the background. The focus is that it would create a password that would automatically clip on to one's clipboard/database for access.

Further Specs

  • What exactly is our app?

    Our app helps users generate passwords for them utilizing ascii as well as numbers and special characters. Because most passwords are around 8-12 characters long, our program makes a complex password and stores it in a database. In doing so, it is meant that the user has access to the generated password that they can copy from their clipboard.

  • How is it implemented in the real world?

    Our app will help users create safer passwords and relieve the stress of having to remember them all because the user would have easy access to their passwords. If they do want to retrieve their password, our app includes a "get password" button by providing the key associated with it. Not only that, the app includes features that the user can quit if wanted, a get help button, and of course but not least the generate password button.

  • Things that we were not able to finish

    One part of the project that we were unable to finish was adding a voice feature to our app. The voice feature was going to be programmed so that users could say their password instead of typing it. This would not only help people who have not learned how to type, such as younger kids. It can also help people with disabilities that find typing to be a challenge for them.
